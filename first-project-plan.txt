- The game: Spaceman

- Variables: CurrentPlayer, Win, gameTimeout
- Data structures: hold a set of items, either letters, shapes, or images, etc. 
- Functions: initiation, checking player status

 
User Stories: 
- The user can start the game by clicking on the letters
- The User can try a specific number of attempts (e.g., 5 - 7)
- The user can see if he guessed the correct words
- The user can see if they lost the round
- The user can start a new round

Other Additions:
- Sound effects during win or lose
- Calm music during game progress
- Click sound effects
- New word pop-up sound effects
